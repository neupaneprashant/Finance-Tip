# MVP Options Monitoring System

This repository is for monitoring options trades and stocks in a portfolio in real time using API from Yahoo Finance, which will act as a prototype for an actual Bloomberg API project.

The original project design is available at https://www.figma.com/design/s42EEnwqrEwHlexyj3PwUC/MVP-Options-Monitoring-System.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
